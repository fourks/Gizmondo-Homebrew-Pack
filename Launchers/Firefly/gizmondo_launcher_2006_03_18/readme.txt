===============================================================================
                               Gizmondo Launcher
===============================================================================

This is a launcher for Gizmondo homebrew applications.

(c) 2006 FireFly

===============================================================================
                               Example Structure
===============================================================================

Here is an example structure for an SD Card with two sub-menus, "games" and
"tools". The "games" menu contains "xrick" and "quake". The "tools" menu
contains "GizmondoCameraCube". The "autorun.exe", "games.exe" and "tools.exe"
files are copies of the launcher. The ".bmp" files are optional.

+ SD Card
|
+-- autorun.exe
|
+-+ games
| |
| +-- games.exe
| |
| +-- games.bmp
| |
| +-+ xrick
| | |
| | +-- xrick.exe
| | |
| | +-- xrick.bmp
| | |
| | +-- ... other xrick files
| |
| +-+ quake
|   |
|   +-- quake.exe
|   |
|   +-- quake.bmp
|   |
|   +-- ... other quake files
|
+-+ tools
| |
| +-- tools.exe
| |
| +-- tools.bmp
| |
| +-+ GizmondoCameraCube
|   |
|   +-- GizmondoCameraCube.exe
